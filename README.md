# Semana-5---Sumativa

## Qué hace este proyecto
Este proyecto en Java es un pequeño sistema para manejar información de personas y empleados. 

Básicamente crea personas, empleados, sus direcciones, y luego imprime todo en la consola. 

También tiene una clase de utilidades con algunos métodos simples como saludo, suma y validar email.

## Paquetes y clases

**org.example.app**
  Main.java: La clase principal. Aquí es donde creamos todos los objetos y mostramos todo en la consola.

**org.example.model**
 Persona.java: Representa a cualquier persona con nombre, apellido, email, teléfono y dirección.
 Empleado.java: Es una persona pero además tiene RUT, cargo y salario.
 Direccion.java: Guarda la información de la dirección (calle, número, ciudad, región, país).

**org.example.util**
  Util.java: Clase con métodos “de ayuda”: saludo, sumar números y validar emails.
